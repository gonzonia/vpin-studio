Selected Game Pinup Media Panel

Virtual Pinball Backup Manager - managing your backups, one game at a time

  Overview

The Selected Game Pinup Media Panel panel provides a read only view of the currently selected games Pinup Popper media is a style consistent with the Pinup Popper Media Manager display. This information mirrors the information found within the Pinup Popper Games Manager.

Users wishing to modify Pinup Popper media seen in this panel should do this through Pinup Popper as the scope of VPBM is limited to the management of game backups rather than editing game data.

The colour of media status buttons reflects the presence of Pinup Media of that type.

  1. A green status button indicates that media was found.
  2. A yellow status button indicates that no media was found.

Clicking on a green media status button will expand the tree view in the Selected Game Content panel to navigate to the media type that has been clicked on.

The source of information viewed in this panel is determined by the selections made in the Selected Game Display Control panel.

  1. If Disk is selected, the displayed content is taken from the currently active Pinup Popper database.
  2. If Backup is selected, the displayed content is taken from the currently games backup file.
