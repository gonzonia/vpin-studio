Selected Game Pinup Details Panel

Virtual Pinball Backup Manager - managing your backups, one game at a time

  Overview

The Selected Game Pinup Details panel provides a read only view of the currently selected games details. This information mirrors the information found within the Pinup Popper Games Manager, but is a read only copy of this information.

Users wishing to modify game data seen in this panel should do this through Pinup Popper as the scope of VPBM is limited to the management of game backups rather than editing game data.

The source of information viewed in this panel is determined by the selections made in the Selected Game Display Control panel.

  1. If Disk is selected, the displayed content is taken from the currently active Pinup Popper database.
  2. If Backup is selected, the displayed content is taken from the currently games backup file.

Content within the Selected Game Pinup Details panel will be colored red if the content differs between the installed game and database, or green if the content is only present in the installed game or database.

  Backup and Restore of Pinup Details

When Pinup details are found to contain differences between the system and the backup, it is possible to perform a backup or restore of just the Pinup details - saving the need to perform another full game backup or restore.

When differences have been detected, the Selected Game Pinup Details panel border is highlighted in an error colour. The examples below show the case where the Game Theme and ROM values are out of synch.

Depending on whether the current or backup filter variables are being displayed, a Backup Values or Restore Values button is provided in the top right corner of the panel to allow the selected games Pinup details to be backed up or restored - as seen below.

Pressing Backup Values or Restore Values button will backup or restore the selected games Pinup details.
