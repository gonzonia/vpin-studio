Selected Game Display Control Panel

Virtual Pinball Backup Manager - managing your backups, one game at a time

  Overview

The Selected Game Display Control panel is provided to allow users to manage what information is displayed for the selected game. The selected game corresponds to the most recently selected game in the Installed Games panels game table.

The control allows the user to select whether game information is to be displayed from the Current system (currently installed game - on disk) or from the Backup. The Backup option is only enabled for games that have a backup file present. By default the Current option is selected.

Content in other selected game content panels is updated to reflect the change and the title of these panels changed to reflect the display source, with the text "(Backup)" added if backup content ids being displayed. Controls in these other panels may be disabled when viewing backups if modification is not possible.

If VPBM has been configiured to allow backup sharing to up to two alternate hosts, information about any alternate hosts set for the selected game will be displayed at the right of screen under the heading Selected Game Backup Shared Hosts.
